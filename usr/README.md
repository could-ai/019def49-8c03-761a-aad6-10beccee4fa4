# Vivo 60 PUBG Sensitivity Guide

A Flutter application providing optimal recommended PUBG Mobile sensitivity settings specifically tailored for Vivo 60 series devices. This app serves as a quick reference guide for players looking to improve their recoil control, aiming precision, and overall gameplay experience on their Vivo smartphones.

## Features
- **Comprehensive Sensitivity Categories:** Covers Free Look Camera, Camera, ADS, Gyroscope, and ADS Gyroscope sensitivities.
- **Tailored for Vivo Devices:** Recommended values designed for the screen response rates and touch sampling rates of modern Vivo phones (like the V60 / Y60 series).
- **Dark Gaming Theme:** A high-contrast, dark mode UI with orange accents, inspired by competitive gaming aesthetics.
- **Easy to Read:** Expansion tiles cleanly organize the extensive list of scope sensitivities (Red Dot, 3x, 4x, 6x, etc.).

## Tech Stack
- **Framework:** Flutter
- **Language:** Dart
- **UI:** Material Design 3

## Setup & Run
1. Ensure you have Flutter installed on your machine.
2. Clone this repository.
3. Run `flutter pub get` to install dependencies.
4. Run `flutter run` to launch the app on your connected device or emulator.
